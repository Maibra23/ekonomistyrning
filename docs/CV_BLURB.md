# CV-punkt

**Ekonomistyrning Sandbox** (Python, Streamlit, Plotly, Qwen3-8B)
Byggde en interaktiv webbapp med fem berakningsmoduler (kalkylering, investeringsbedomning, budgetering, standardkostnadsanalys, kunskapstest) och en LLM-tutor som forklarar resultat grundat i anvandarens siffror. Inkluderar Monte Carlo-simulering, Excel-export och deterministisk fallback vid LLM-avbrott.
